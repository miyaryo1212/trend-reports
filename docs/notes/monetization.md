# マネタイズ構想（trend-reports）

**ステータス**: 構想段階 / `develop` ブランチで検討中
**最終更新**: 2026-04-22

## 方針

- **あくまで「おまけ」** — 本気で稼ぐ気はない、運営継続の足しになれば十分
- **ペナルティ回避を最優先** — AdSense規約違反・プライバシー法令違反は絶対NG
- **主張しすぎない配置** — 読書体験を壊さない範囲で、アクセシビリティ配慮
- Claudeブランドのオレンジ系トーンは維持

## 現状確認

- Astro 構成の静的サイト
- ページ種別: TBD（トップ / ランキング / 各Report / Updates）
- 想定アクセス層: AI/技術系の読者層、ボリュームはGA導入後に実測

## Phase 1: 計測・同意基盤

### GA4
- gtag 直埋め or GTM 経由（要検討）
- 新規プロパティ作成

### Cookie 同意バナー
- **Consent Mode v2 対応必須**（EU圏アクセスで違反回避）
- 同意前はGA発火させない
- 実装候補: 自作 / CMP（Cookiebot, Klaro 等）
- TBD: 自作 vs ツール導入

### プライバシーポリシー / Cookieポリシーページ
- `/privacy`, `/cookies` を新設
- AdSense申請時に必須

## Phase 2: AdSense

### 審査通過条件
- プライバシーポリシー掲載
- 一定のコンテンツ量（毎朝Report更新で蓄積中）
- オリジナリティ（Claudeキュレーションによる独自観点）

### 配置設計
- **デスクトップ**: 左下の空きスペースに控えめに
- **モバイル**: ボトム sticky（よくある形式）
- **誤タップ防止**: 操作要素との距離を十分に確保
- **アクセシビリティ**: スクリーンリーダーでスキップ可能、コントラスト配慮

## ペナルティ回避チェックリスト

- [ ] 自己クリック禁止（広告表示中は触らない運用ルール）
- [ ] 家族・知人にクリック依頼しない
- [ ] トラフィック水増し工作しない
- [ ] 操作要素との距離を十分に空ける
- [ ] プライバシー/Cookieポリシー明示
- [ ] EU Consent Mode v2 準拠

## KPI / 計測

- PV（GA4）
- CTR / RPM（AdSenseダッシュボード）
- 頻度: 週次チェック程度
- 期待値はあえて低く（おまけ路線）
