# マネタイズ構想（trend-reports）

**ステータス**: Phase 1 実装完了 / Phase 2 審査申請準備完了
**最終更新**: 2026-04-25

## 方針

- **あくまで「おまけ」** — 本気で稼ぐ気はない、運営継続の足しになれば十分
- **ペナルティ回避を最優先** — AdSense規約違反・プライバシー法令違反は絶対NG
- **主張しすぎない配置** — 読書体験を壊さない範囲で、アクセシビリティ配慮
- Claudeブランドのオレンジ系トーンは維持

## Phase 1: 計測・同意基盤 ✅ 完了

- **GA4**: gtag 直埋め（GTMなし）。`PUBLIC_GA_MEASUREMENT_ID` 未設定時はスクリプトごとロードしない
- **Cookie 同意バナー**: 自作（CMPなし）、Consent Mode v2 対応、localStorage に状態保存
- **プライバシーポリシー**: `/privacy`
- **Cookieポリシー**: `/cookies`（同意リセットボタン付き）

**残アクション（ユーザー側）**: GA4プロパティ作成 → `PUBLIC_GA_MEASUREMENT_ID` を本番env varに設定

## Phase 2: AdSense ⏳ 審査申請準備完了

### 申請準備実装内容 ✅
- `<script async src="...adsbygoogle.js?client=...">` を head 条件注入（`PUBLIC_ADSENSE_CLIENT_ID` 制御）
- `/ads.txt` エンドポイント（env var から動的生成）
- プライバシー/Cookieポリシー記述を「準備中」に更新

**残アクション（ユーザー側）**:
1. adsense.google.com でアカウント作成 → サイト追加
2. `ca-pub-XXXXXXXXXXXXXXXX` 取得
3. `PUBLIC_ADSENSE_CLIENT_ID` を本番env varに設定
4. デプロイ反映後、申請ボタンを押す
5. 審査通過（数日〜数週間）を待つ

## ペナルティ回避チェックリスト

- [ ] 自己クリック禁止（広告表示中は触らない運用ルール）
- [ ] 家族・知人にクリック依頼しない
- [ ] トラフィック水増し工作しない
- [ ] 操作要素との距離を十分に空ける
- [x] プライバシー/Cookieポリシー明示
- [x] EU Consent Mode v2 準拠

## KPI / 計測

- PV（GA4）
- CTR / RPM（AdSenseダッシュボード）
- 頻度: 週次チェック程度
- 期待値はあえて低く（おまけ路線）
