## 公式アップデート

### OpenAI GPT-5.4

OpenAIが最新モデルGPT-5.4を発表。サイトマップで新規ページが検出された段階の情報。

### OpenAI GPT-5.3 Codex Spark

コード生成特化の新モデルGPT-5.3 Codex Sparkを発表。超高速推論が特徴で、Cerebras専用ハードウェアでの動作が報告されている。

### OpenAI GPT-5.2 Codex

コーディング向けモデルGPT-5.2 Codexをリリース。

### ChatGPT広告機能

OpenAIがChatGPTへの広告導入の新たな購入方法を発表。Ads Manager βの拡大、CPC対応・最低予算なしのセルフサーブ広告を全米広告主向けに解放。

### ChatGPTパーソナルファイナンス機能

ChatGPTに個人資産管理・家計機能を追加。Plaid連携による銀行・証券口座管理やGPT-5.5による財務分析が可能に（米国Proユーザー限定）。（前回から継続）

### ChatGPT商品発見機能

ChatGPT内でのショッピング・商品検索機能を強化。

### OpenAI Model Spec更新

OpenAIがモデルの行動指針「Model Spec」の最新版を公開。（前回から継続）

### OpenAI AIが離散幾何学の予想を反証

OpenAIの内部汎用推論モデルが離散幾何学の中心的な未解決問題（平面単位距離問題）を自律的に反証し、80年来の常識を覆す新構成を発見。（前回から継続）

### Anthropic Project Glasswing

AWS・Apple・Microsoft・Google等12社参加のAIセキュリティイニシアチブ。Claude Mythos Previewモデルで脆弱性発見を加速し、1億ドルのクレジットを提供。（前回から継続）

### Anthropic Claude火星探査

ClaudeがNASAパーサヴィアランスの火星走行経路を自動生成し、経路計画時間を約50%削減。

### Anthropic Project Deal

AIエージェント間マーケットプレイス取引実験を発表。高性能モデル使用者がより有利な取引結果を得ることを実証。

### Anthropic Academy

Claude Code実践ガイドやAPI開発ガイド等を提供する新学習プラットフォームを公開。

### Anthropic プロモーションクレジット制度

有料プラン向けの利用クレジット提供条件を新設（2026年5月19日付）。

### OpenAI ChatGPT Images 2.0

ChatGPTの画像生成機能の大型アップデート版を発表。
