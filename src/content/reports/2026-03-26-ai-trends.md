## 公式アップデート

### OpenAI Model Spec公開

モデル行動の公開フレームワーク「Model Spec」についての解説記事を公開。安全性・ユーザーの自由・説明責任のバランスに関する指針を文書化している。

[ソース](https://openai.com/index/our-approach-to-the-model-spec)

### OpenAI Safety Bug Bountyプログラム開始

AIの悪用・安全リスクの発見を目的とした新しいバグバウンティプログラムを発表。エージェント脆弱性、プロンプトインジェクション、データ流出が対象範囲に含まれる。

[ソース](https://openai.com/index/safety-bug-bounty)

### OpenAI gpt-oss-safeguard公開

開発者向けに、10代ユーザーの安全ポリシーをプロンプトベースで提供するOSSツール「gpt-oss-safeguard」を公開。年齢に応じたリスク軽減をAIシステムに組み込むことを支援する。

[ソース](https://openai.com/index/teen-safety-policies-gpt-oss-safeguard)

### OpenAI Foundation 10億ドル投資計画

OpenAI Foundationが疾病治療・経済機会・AIレジリエンス・コミュニティプログラムの4分野に少なくとも10億ドルの投資を行う計画を発表。

[ソース](https://openai.com/index/update-on-the-openai-foundation)

### ChatGPT Agentic Commerce Protocol

ChatGPTにリッチなショッピング体験を導入。Agentic Commerce Protocolにより、商品発見・サイドバイサイド比較・マーチャント連携をエージェント型で実現する。

[ソース](https://openai.com/index/powering-product-discovery-in-chatgpt)

### Sora 2の安全対策

Sora 2とSoraアプリの安全対策について公開。動画生成モデル特有の安全課題に対し、基盤レベルでの具体的な保護策を構築している。

[ソース](https://openai.com/index/creating-with-sora-safely)

### Anthropic Claude Code auto mode

Claude Codeに許可承認の93%を自動化する新モード「auto mode」が追加。プロンプトインジェクション検知とアクション分類器の2層防御を搭載。`claude --permission-mode auto`で起動する。

### Anthropic Claude Code自律作業強化

Claude Codeにおけるコンピュータ操作を含む自律的な作業能力の向上が発表された。iMessage対応によるスマホからのPC操作も可能に。
